# 🚗 Blitz Rent a Car - Changelog

## Версия 1.1 - 2026-07-14

### ✅ Внесенные изменения

#### 1. **Hero секция на весь экран**
- Изменена высота с `600px` на `100vh` (весь экран)
- Адаптация для всех разрешений (desktop, tablet, mobile)
- Hero теперь занимает полный экран на всех устройствах

#### 2. **Мобильная версия модального окна**
- **Fullscreen режим** на мобильных устройствах
- Модальное окно занимает весь экран (100vh × 100%)
- Улучшенная кнопка закрытия (темный фон, лучшая видимость)
- Убраны скругления на мобильных для максимального использования пространства
- Оптимизированные отступы (16px вместо 20px)

#### 3. **Умная кнопка "Забронировать"**
- **На мобильных**: прямой звонок `tel:+359988746138`
- **На десктопе**: диалог с выбором способа связи:
  - WhatsApp
  - Viber
  - Telegram
  - Email
  - Звонок
- Автоопределение устройства через User Agent

#### 4. **Упрощенные названия сезонов**
- Было: "Высокий (май-сентябрь)" → Стало: "Сезон"
- Было: "Низкий (октябрь-апрель)" → Стало: "Не сезон"
- Изменения для всех 3 языков (RU/EN/BG)
